Heritage Aroma is a fully functional PHP & MySQL based Ecommerce Website that allows users to browse products, add them to the cart, manage multiple delivery addresses, complete checkout and payment, and track their orders.
The project is designed to reflect a real-world ecommerce workflow with proper authentication, session handling, and order management.
